# Landing Page

Simple static landing page with:
- Header with logo and navigation
- Hero section (title, text, signup button, image)
- Info section with four evenly sized cards
- Quote section with attribution
- Call-to-action and footer

## Run Locally
Open `index.html` directly in a browser, or start a local server:

```bash
cd "c:/Users/hp/OneDrive/Desktop/HOME/repo/landing-page"

# Then open: http://localhost:8000/landing-page/index.html
```

## Files
- `index.html` — page markup
- `style.css` — styles (flex layout for info cards; hero image sized via `.hero-img`)
- `resources/` — images (e.g., `01.png`, `02.png`)

## Notes
- Hero image uses `.hero-img` with fixed width/height .
- Quote attribution uses class `new` for styling.
